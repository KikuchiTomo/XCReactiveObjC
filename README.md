# XCReactiveObjC
## 概要
ReativeObjCとReactiveObjCBridgeをSwift Package Managerで使用するためのリポジトリ

## 使い方
それぞれ以下の設定値を利用してください
なお, `ReactiveObjCBridge`の方を追加すれば, `ReactiveObjC`も自動で追加されます
#### ReactiveObjC
- url: `https://github.com/KikuchiTomo/XCReactiveObjC.git`
- branch: `ReactiveObjC`

#### ReactiveObjC
- url: `https://github.com/KikuchiTomo/XCReactiveObjC.git`
- branch: `ReactiveObjC`
